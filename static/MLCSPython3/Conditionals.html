Conditionals

Welcome to learning about conditionals in Python 3! Here you will be learning about if statements, else statements, comparators, and boolean operators!

If Statements

Now that we've learned how to create variables, let's learn a way to use them. Conditional if statements are the code equilavent of if _____ happens, then _____.

Ex.
x = 5
if x > 4:
    x = x + 1
print(x)

Output:
6

What happened above is that x was set to the value of 5, and then the if statement asked if x is greater than 4, then add 1 to x. > means greater than, < means less then, and == means equals to. Only use == in if statements. We will get into more complex comparators like these laters, but these are the basics and are almost always used when concerning numbers. If x was 3 instead of 5 in the example above then x would print out the number 3. The code would not add 1 to x, for x was not greater than 4. Also if x had the value of 4 it would print out 4, for 4 is not greater than 4.

Elif and Else Statements

Hello

What if you want to have more than one conditional statement? Sure you can add another if statement, but while running it will take both if statements in at once. If you want to traverse each if statement one at a time, that's when elif and else statements come in.

Basically elif statements go after if statements. The code runs and sees if the conditional given in the statement is correct. If it's false or else if  then and only then will it consider the next elif statement. Look at the example below.

Ex.
if 3 > 4:
    print("hi")
elif 2 == 2:
    print("goodbye")

Output:
goodbye

Only after considering the first if statement will it start considering the second elif statement. However if the first conditional is true, it will ignore anything that comes after it when using elif.

Ex.
if 5 > 4:
    print("hi")
elif 2 == 2:
    print("goodbye")

Output:
hi

If the elif statement was just a normal if statement then it would've printed both hi and goodbye. Elif statements are only read if the conditional statement above it is false. 

In cases when all the conditional cases are false, then the else statement comes in. 

Ex.
if 3 > 4:
    print("hi")
elif 3 == 2:
    print("goodbye")
else:
    print("It failed")

Output:
It failed

The else statement only goes through if all conditionals above are not true. This can be useful when getting unexpected results.

Comparators

Earlier you learned the comparators greater than (>), less than (<), and equal to (==). Now we will be learning more comparators. The next ones we will be learning about also mainly apply to numbers. They are greater than or equal to (>=), less than or equal to (<=), or not equal to (!=). Some examples are below.

Ex.
if 4 != 3:
    print("not equal to")
if 10 >= 6:
    print("greater than or equal to")
if 3 <= 3:
    print("less than or equal to")

Output:
not equal to
greater than or equal to
less than or equal to

You can also use == and != to see if strings are equal, and to see if boolean variables are True or False. Look at the example below.

Ex.
x = False
if True:
    print("True!")
if x == False:
    print("False")

Output:
True
False

Also if you want want one condition or another condition to be true, you use the or command. If you want multiple conditions to be true at once, you use the and command.

Ex.
x = False
y = 5
z = "Hi"
if z == "Hi" and x == False:
    print("Both!")
if y < 4 or x == False:
    print("Either!")

Output:
Both!
Either!

Even though y, which held the value 5, was not less then 4, x did equal False, and if either of those conditions are met then the if statement continues. For the and command both conditions must be met for the if statement to continue.

Nested If Statements

What are nested if statements? Nested if statements are basically an if statement in another if statement. An example is below.

Ex.
x = 5 
y = 3
if x > 4:
    if y == 3:
        print("True!")

Ouput:
True!

This basically says if a conditional is true, then ask another conditional. If this conditional is true do an action. The logic is similar to using the and command, however you can add actions in between nested if loops, so only a certain amount of conditions need to be met to do certain actions. When using and, all conditions must be met. An example is below.

Ex.
x = 2
y = 4
z = 1
if x == 2:
    print("hi")
    if y > 3:
        print("goodbye")
        if z == 0:
            print("hello again")

Output:
hi
goodbye

Only hi and goodbye were printed for their conditions were met, if we use an and command instead nothing would've printed for all the conditions are not true, and if we did an or command everything would've printed for at least one condition was met. Nested if statements let you pick and choose what and how many conditions must be made for an action to go through.
