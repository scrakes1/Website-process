How To Install Idle Python 3

How to download Idle Python 3 is quite simple. You must go to the official python.org website and go to downloads and download the latest version. They have options to download it on all different type of computers and links to go there. First go to this website and click download the latest version.

Click continue and agree and install once it finishes downloading the installation for python. After it is finished installing Python, it will open a python folder in your finder. In the folder will the Idle app. Open the Idle app and you will be able to code Python 3!
